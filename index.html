Gif Generator

When the user types in a topic and hits search it will add a button to the list. When the user clicks on a button, the page will grab 10 static, non-animated gif images from the GIPHY API and place them on the page. If the user clicks one of the still GIPHY images, the gif will animate. If the user clicks the gif again, it will stop playing.

Check it out on Github!
